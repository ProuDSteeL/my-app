# Feature Research

**Domain:** Book summary / non-fiction reading apps (Russian market)
**Researched:** 2026-03-18
**Confidence:** HIGH (based on analysis of Blinkist, Smart Reading, Shortform, getAbstract + domain expertise)

## Feature Landscape

### Table Stakes (Users Expect These)

| Feature | Why Expected | Complexity | Notes |
|---------|-------------|------------|-------|
| Каталог книг с поиском и фильтрами | Базовая навигация по контенту, без этого пользователь не найдёт нужное | Medium | Полнотекстовый поиск, фильтры по категориям, debounce, история поиска |
| Текстовые саммари с ридером | Ядро продукта — у всех конкурентов это есть | Medium | Markdown-рендеринг, настройки шрифта/темы/размера, оглавление |
| Аудиоверсии саммари | Blinkist, Smart Reading, Shortform — все имеют аудио. Пользователи ожидают | High | Мини-плеер + полноэкранный, скорость 0.5x–2.0x, таймер сна, фоновое воспроизведение |
| Авторизация (email + OAuth) | Стандарт для любого SaaS-приложения | Low | Email + пароль, Google OAuth. Apple Sign-In не нужен для web v1 |
| Freemium-модель с подпиской | Все конкуренты: бесплатный доступ к части контента + платная подписка | Medium | 5 бесплатных саммари, превью 20% для остальных, Pro для полного доступа |
| Прогресс чтения | Пользователь должен видеть, где остановился | Low | Синхронизация позиции между сессиями |
| Полки / избранное | Базовое управление библиотекой есть у всех | Low | Избранное / Прочитано / Хочу прочитать |
| Профиль пользователя | Стандартная функция, привязка подписки | Low | Аватар, тип подписки, базовая статистика |
| Категории и тематические подборки | Все конкуренты группируют контент по темам | Low | Бизнес, психология, саморазвитие, наука и т.д. |
| Тёмная тема | Современный стандарт, особенно для ридеров | Low | Светлая + тёмная тема приложения, отдельные темы ридера |
| Страница саммари (карточка книги) | Пользователь должен понять, стоит ли читать | Low | Обложка, мета, теги, «О книге», «Зачем читать?», «Об авторе» |

### Differentiators (Competitive Advantage)

| Feature | Value Proposition | Complexity | Notes |
|---------|-------------------|------------|-------|
| **Полный офлайн-доступ (текст + аудио)** | Smart Reading имеет офлайн, но ненадёжный. Blinkist — только аудио. Наш: полноценное скачивание текста и аудио | High | Hive для локального кеша, Service Worker для PWA, лимит хранилища, автоочистка. **Главный дифференциатор** |
| **Цена 2-3x ниже Smart Reading** | Smart Reading ~500-700₽/мес, наш Pro — 200₽/мес / 1200₽/год | Low (бизнес-решение) | Возможно за счёт меньшей команды и ручного создания контента автором |
| **«Warm Brutalism» дизайн** | Уникальная визуальная идентичность vs. generic UI конкурентов | Medium | Терракотовая палитра, Playfair Display, жёсткие рамки, книжный стиль. Запоминаемость бренда |
| Карточки ключевых идей | Shortform имеет exercises, но карточки — более визуальный формат | Medium | Карусель ключевых идей на странице саммари, быстрый обзор без полного чтения |
| Выделение текста + сохранение цитат | Shortform имеет highlights, Smart Reading — нет. Цветовая маркировка добавляет ценность | Medium | Цветовая маркировка, группировка по книгам, экспорт. Зависит от ридера |
| Персональные рекомендации (Pro) | Smart Reading: подборки от экспертов. Наш: алгоритмические рекомендации на основе истории | Medium | Для Pro-пользователей, на основе истории чтения. Не AI — простая логика «похожие книги» |
| PWA с Service Worker | Web-first подход позволяет обойтись без магазинов приложений, мгновенный доступ | Medium | Нет комиссии 30% Apple/Google, быстрый запуск, установка на домашний экран |
| YooMoney (удобство для RU) | Конкуренты часто привязаны к картам. YooMoney — кошелёк, карты, SBP | Low | Единственный платёжный метод для v1, физлица |

### Anti-Features (Commonly Requested, Often Problematic)

| Feature | Why Requested | Why Problematic | Alternative |
|---------|---------------|-----------------|-------------|
| AI-генерация саммари | Масштабирование контента | Низкое качество, юридические риски, потеря авторского голоса. Smart Reading тоже делает вручную | Ручное создание с контролем качества |
| Социальные функции (лента, комментарии) | «Вовлечённость», retention | Требует модерации, отвлекает от ядра, сложная инфраструктура, малая аудитория на старте | Экспорт цитат в соцсети (v2+) |
| Геймификация (ачивки, бейджи, уровни) | Удержание пользователей | Может восприниматься как навязчивость, усложняет UX, не связана с ценностью продукта | Простая статистика (книги, дни подряд, время) |
| Real-time чат / книжный клуб | Комьюнити | Высокая сложность, модерация, серверные расходы, не относится к ядру | Нет для v1, возможно v3+ |
| Видео-саммари | TikTok-формат популярен | Огромные затраты на производство и хранение, не масштабируется одним автором | Аудио + текст покрывают потребности |
| AI-чатбот «спроси о книге» | Тренд в Blinkist (Blinkist AI) | Высокая стоимость API, галлюцинации, юридические вопросы, отвлекает от чтения | Хорошо структурированные саммари с ключевыми идеями |
| Push-уведомления (агрессивные) | «Возвращение пользователей» | Раздражают, ведут к удалению. Особенно плохо для PWA (ненадёжная доставка) | Ненавязчивые email-дайджесты (v2+) |
| Интеграция с LMS/корпоративные планы | getAbstract так делает | Совершенно другой сегмент (B2B), требует отдельной команды продаж | Фокус на B2C, корпоративные планы — v3+ |
| A/B тестирование paywall | Оптимизация конверсии | Преждевременная оптимизация, нужна значимая аудитория | Одна проверенная стратегия paywall для v1 |
| Тесты / квизы после чтения | Smart Reading имеет, повышает retention | Дополнительный контент на каждую книгу, время на создание, сомнительная ценность для взрослой аудитории | Карточки ключевых идей как быстрое повторение |
