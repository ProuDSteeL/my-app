# BookSummary — Саммари нон-фикшн книг

## What This Is

Кросс-платформенное приложение для чтения саммари нон-фикшн книг на русском языке, построенное на Flutter. Пользователи получают доступ к текстовым саммари, аудиоверсиям и карточкам ключевых идей с возможностью скачивания для полноценного офлайн-доступа. Контент создаётся вручную автором проекта и загружается через защищённую админ-панель внутри приложения. V1 — Web/PWA, мобильные приложения (iOS, Android) планируются позже с той же кодовой базой Flutter.

## Core Value

Полноценный офлайн-доступ к качественным саммари нон-фикшн книг на русском языке по цене в 2–3 раза ниже конкурентов.

## Context

### Целевая аудитория

- Предприниматели и менеджеры 25–45 лет
- Студенты и самоучки, стремящиеся к развитию
- Люди без времени читать целые книги
- Русскоязычная аудитория, не владеющая английским для Blinkist/Shortform

### Конкурентная среда

- Smart Reading — основной конкурент на русском рынке, цены выше в 2–3 раза
- Blinkist, Shortform — англоязычные аналоги, недоступны целевой аудитории
- Ключевое отличие: полноценный офлайн-доступ (текст + аудио), низкая цена, нативный Flutter UX

### Дизайн-концепция: «Warm Brutalism»

Жёсткий, графичный стиль с акцентом на типографику и контраст. Вдохновение: Apple Books + Readwise + брутализм в вебе.

**Палитра (светлая тема):**
- Primary (Акцент): Терракотовый `#C05621`
- Primary variant: Тёмная глина `#9C4221`
- Secondary: Тёплый жёлтый `#D69E2E`
- Background: Кремовый `#FFFAF0`
- Surface: Песочный `#FEFCF3`
- On Background: Почти чёрный `#1A202C`
- On Surface: Тёплый серый `#718096`
- Card border: Песочная рамка `#E2D8C3`

**Палитра (тёмная тема):**
- Background: Тёмный графит `#1A1A2E`
- Surface: `#232340`
- Primary: Светлая терракота `#ED8936`
- On Background: Кремовый `#FEFCF3`
- On Surface: Приглушённый `#A0AEC0`

**Типографика:**
- Заголовки: Playfair Display (Bold/SemiBold)
- Основной текст: Source Sans 3 (Regular/SemiBold)
- Текст чтения: Source Serif 4 (настраиваемый 18–24sp)
- UI-элементы: JetBrains Mono (счётчики, таймеры)

**Компоненты:**
- Карточки: прямоугольные (borderRadius: 4px), тонкая рамка вместо теней, сдвиг при нажатии
- Кнопки: плоские без теней, primary залитые терракотовым (6px), secondary outline
- Chips: прямоугольные (4px), активный залит терракотовым
- Навигация: нижняя панель 5 вкладок, line-иконки, без тени, верхняя разделительная линия
- Иконки: Phosphor Icons (thin 1.5px stroke, fill при выборе)
- Анимации: SharedAxisTransition, FadeIn + SlideUp, skeleton тёплого песочного цвета

### Контент

## Constraints

- **Tech stack**: Flutter (Web/PWA для v1) + Riverpod + GoRouter + Supabase + Hive + just_audio
- **Backend**: Supabase (Auth, PostgreSQL с RLS, Storage, Edge Functions)
- **Платежи**: Только YooMoney для физлиц (Edge Function webhook + redirect flow)
- **Язык**: Русский (основной), подготовка к i18n
- **Производительность Web**: первая загрузка < 3 сек на 3G, bundle < 5 МБ gzipped
- **Безопасность**: RLS на все таблицы, JWT, валидация файлов, Markdown sanitization, HTTPS only
- **Доступность**: WCAG AA (контраст 4.5:1), semantic labels, масштабирование шрифтов
